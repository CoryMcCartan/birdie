/***********************************************************************************
* name:     layout_ncvoter.txt
* purpose:  Contains all legally available voter specific information. Personally 
*           identifying information (PII) such as birth date and drivers license are
*           not included. Voter registrations with a voter_status_desc of 'Removed' 
*           are omitted whenever the most recent last voted date is greater than 10 years.
*           This is a weekly point-in-time snapshot current per file date and time.
* updated:  09/07/2021
* format:   tab delimited
* instructions: 
*            1) extract using a file archiving and compression program (eg. WinZip)
*            2) can be linked to ncvhis file by ncid
***********************************************************************************/
